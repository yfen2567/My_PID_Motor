# My_PID_Motor v2.1 LD-003 负载扰动测试报告

## 1. 测试结论

结论：**PASS（轻载趋势初步通过，扰动幅度有限）**

baseline 阶段在 target=700 下稳定运行；施加轻载后 actual 均值由 690.7 降至 671.2，PWM 均值由 139.7 升至 140.6；`RELEASE_LOAD_CONFIRMED` 后 actual/PWM 保持在 baseline 范围内；全程 `fault:NONE`，stop 后安全归零，`get fault` 返回 `FAULT_SNAPSHOT_NONE`。

`run=1` 后、`BASELINE_START` 前存在一条 `actual=1200` 的启动瞬态。本报告完整保留该点，但没有把它混入 baseline 统计。负载阶段变化幅度较小，因此结论仅限较高目标速度下的基本轻载补偿趋势，不代表高性能或精确动态控制验证。

## 12. 异常与边界说明

### 本次测试能证明

- 排除启动瞬态后，target=700 的 baseline 能持续运行；
- 本次轻载期间 actual/PWM 呈现方向正确但幅度有限的补偿趋势；
- 释放确认后系统保持运行并回到 baseline 取值范围；
- 全程未记录 fault、持续停速、串口中断或 MCU 复位；
- stop 后软件状态和 PWM 命令安全归零；
- 无历史故障快照时返回 `FAULT_SNAPSHOT_NONE`。

### 本次测试不能证明

- `actual=1200` 启动瞬态的物理转速真实性或根因；
- 人工负载的精确大小、转矩和三次测试间的一致性；
- 精确恢复时间、控制带宽、频响或严格动态性能；
- 物理 PWM 波形、驱动电流、供电压降或编码器 A/B 相电气质量；
- 其他电机、驱动板、供电、负载和参数条件会得到相同结果；
- 长期可靠性或全部异常场景已经覆盖。

原始日志没有记录具体脚本文件名、固件 commit 或构建产物 hash，这些仍是 v2.1 证据链缺口。

按当前“500/600/700 三档人工轻载趋势验证”的范围，LD-001、LD-002、LD-003 已各自形成原始日志和正式报告，可以收口。当前没有证据要求为了同一目的额外复测；若未来需要定量动态指标，应另行定义可测量负载和更严格测试方法，不能由本轮人工轻载数据外推。
